DOCUMENTATION
Andika Basics
========================

This file describes the documentation files included with the Andika 
Basics font family. This information should be distributed along with the 
Andika Basics fonts and any derivative works.

Primary documentation for the Andika Basics font family is on the Andika 
website (http://scripts.sil.org/Andika_Basics).

The AndikaBasics-features.pdf file illustrates the use of OpenType and 
Graphite features.

The AndikaBasics-features.odt file illustrates the use of OpenType and 
Graphite features (source file for the pdf).
