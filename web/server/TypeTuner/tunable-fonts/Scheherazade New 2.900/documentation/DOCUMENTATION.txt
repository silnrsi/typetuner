DOCUMENTATION
Scheherazade
========================

This file describes the documentation files included with the Scheherazade 
font. This information should be distributed along with the 
Scheherazade font and any derivative works.

Primary documentation for the Scheherazade font is on the Scheherazade 
website (http://software.sil.org/scheherazade/).

The Scheherazade-features.pdf file illustrates the use of OpenType and 
Graphite features.

The Scheherazade-features.odt file illustrates the use of OpenType and 
Graphite features (source file for the pdf).

The Scheherazade-typesample.pdf file gives a sample of all the characters in 
the font.

The Scheherazade-typesample.odt file gives a sample of all the characters in 
the font (source file for the pdf).
