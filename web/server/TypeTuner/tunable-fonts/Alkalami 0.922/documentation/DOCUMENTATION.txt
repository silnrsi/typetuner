DOCUMENTATION
Alkalami
========================

This file describes the documentation files included with the Alkalami 
font family. This information should be distributed along with the 
Alkalami fonts and any derivative works.

Primary documentation for the Alkalami font family is on the Alkalami 
website (does not exist yet).

The Alkalami-features.pdf file illustrates the use of OpenType features.

The Alkalami-features.indd file illustrates the use of OpenType features 
(source file for the pdf).

The Alkalami-typesample.pdf file gives a sample of all the characters in 
the font.

The Alkalami-typesample.indd file gives a sample of all the characters in 
the font (source file for the pdf).
