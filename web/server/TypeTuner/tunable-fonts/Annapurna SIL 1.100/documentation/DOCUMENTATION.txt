DOCUMENTATION
Annapurna SIL v1.100
========================


This file describes the documentation files included with the Annapurna SIL
font. This information should be distributed along with the Annapurna SIL
font and any derivative works.

Primary documentation for the Annapurna SIL font is on the Annapurna SIL
website (http://scripts.sil.org/AnnapurnaSIL).

Annapurna SIL TypeTuner guide.pdf - documents the tunable features to
guide the user in the selection of variant glyphs for a new font.

AnnapurnaSIL-features.pdf - documents the OpenType and Graphite
features, many for use in minority languages, in the font.

AnnapurnaSIL-GraphiteFeatures.odt - documents the Graphite features in the
font. It serves to illustrate the use of Graphite features in OpenOffice 3.2+
and LibreOffice 3.4+.
